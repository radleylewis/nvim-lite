# nvim-lite
A suckless NeoVim Config

Requires NeoVim 0.11.2 or later.

Copy and enjoy it with:
```bash
mkdir -p ~/.config/nvim && curl -fsSL https://raw.githubusercontent.com/radleylewis/nvim-lite/master/init.lua -o ~/.config/nvim/init.lua
```
